function isSubsetOf (otherobj)
  For each myp in my properties, named mypn:
    if otherobj has property otherp named mypn then:
      if (myp is not otherp) then:
        return false
    else return false

  FOR each myo in my objects, named myoname :
    if otherobj has an obj othero named myoname then: 
      if (myo.isSubsetOf(othero) is false) then:
        return false
    else: 
      return false

  FOR each myr names myrname in my relations:
    if otherobj has a relation otherr named myrname then:
      if (myr.isSubsetOf(otherr)) then:
        return false
    else:
      return false

  return true
